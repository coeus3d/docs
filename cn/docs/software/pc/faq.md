# 常见问题

## 在线设备数量不够

PC软件菜单栏选择连接->断开->连接，或者点击在线设备更新

### 问题依旧？

请将设备和路由断电重启，并重新连接

## 路由上传  

### 上传提示上传失败 

切割软件菜单栏选择连接->断开->连接，查看在线设备数量，如无设备出现，请将设备和路由断电重启并连接


### 设备始终无法连上路由

手机搜索设备WiFi，查看播放列表是否正常刷出，如不能刷出请重启设备

### 长时间无操作后，无法上传文件 

请重启并重新连接设备

## TF 卡操作

### 提示上传完成，播放列表无显示，也不能播放

上传前先点击 TF 列表，查看列表是否能刷新出来

如果刷新不出播放列表：

- 拨出 TF 卡
- 擦去 TF 金手指接触位置灰尘
- 重新插回路由器

!!! note "擦拭金手指"
    可以使用橡皮擦轻轻擦拭

### 上传速度慢

请采用网线直连方式

### 路由器 TF 卡列表无法刷出

请联系技术人员解决
